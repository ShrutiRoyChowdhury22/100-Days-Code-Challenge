# Question:
	 Write a program to find the factorial of any number entered by the user.

# Programming Language:
		     JAVA

# Input:
      Any  positive integer n.

# Output:
       Print the factorial of the number which is entered by user.

# Test Cases:
	   Case 1:
		  n = 5
		    print = 120
	   Case 2:
		  n = 7
		     print = 5040
	   Case 3:
		  n = 10
      		    print = 3628800

# Logic:
      The value of factorial will be store in f. 
	If , n = 5
f = f*i(where i<=n)
	f = 1*1=1
	f = 1*2=2
	f = 2*3=6
	f = 6*4=24
	f = 24*5=120
	
	If, n = 7
f = f*i(where i<=n)
	f=1*1=1
	f=1*2=2
	f=2*3=6
	f=6*4=24
	f=24*5=120
	f=120*6=720
	f=720*7=5040

	If, n = 10
f = f*i(where i<=n)
	f=1*1=1
	f=1*2=2
	f=2*3=6
	f=6*4=24
	f=24*5=120
	f=120*6=720
	f=720*7=5040
	f=5040*8=40320
	f=40320*9=362880
	f=362880*10=3628800
